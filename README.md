# CourseMate

**CourseMate** is a website that has collection of online learning resources for programming. This repo contains back-end for CourseMate.

## Tech Used

#### Back-end

-   Node.js (with TypeScript)
-   Express
-   MongoDB
-   Mongoose
-   Passport
-   Google OAuth

#### Testing

-   Supertest
-   Jest

#### CI

-   Travis CI
